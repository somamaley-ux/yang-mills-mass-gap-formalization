\begin{theorem}[Euclidean exponential clustering $\Rightarrow$ spectral gap (Hilbert-space form)]\label{thm:R1_HS}
Fix $d=4$ and a compact connected simple gauge group $G$, together with the fixed finite-dimensional unitary Wilson representation $\rho$ used in Theorem~\ref{thm:main}.
Fix the same weak-window target coupling $u_{\mathrm{ren}}\in(0,u_\ast]$ as in Theorem~\ref{thm:main}. Let $\omega^{(u_{\mathrm{ren}})}$ be the full sharp-local continuum OS-positive state on $\mathcal A_+$ furnished by Corollary~\ref{cor:LaneA_full_scope} and the tuned-dyadic bounded base state of Theorem~\ref{thm:continuum_tightness}. Then OS reconstruction for $\omega^{(u_{\mathrm{ren}})}$ yields $(\mathcal H,\Omega,H)$.

Assume that for this same state there exists a dense positive-time probe core $\mathcal G_{\mathrm{ent}}\subset\mathcal A_+$, a dense unbounded time set $\mathbb D\subset[0,\infty)$ with $0\in\mathbb D$, and a constructive rate $m=\mathfrak m_\ast>0$ such that the centered OS-classes of $\mathcal G_{\mathrm{ent}}$ are dense in $\Omega^\perp$ and
\[
\big|\omega^{(u_{\mathrm{ren}})}(\Theta\widetilde F\,\tau_s\widetilde F)\big|\le C_F e^{-\mathfrak m_\ast s}
\qquad (F\in\mathcal G_{\mathrm{ent}},\ s\in\mathbb D)
\]
for suitable constants $C_F<\infty$.
Then
\begin{equation}\label{eq:R1_gap}
\mathrm{Spec}(H)\subset\{0\}\cup[\mathfrak m_\ast,\infty),\qquad \ker(H)=\mathbb C\Omega.
\end{equation}
\end{theorem}

The Route~1 source of this input is Appendix~F via Theorem~\ref{thm:route1_E2_internal},

Feeding that verified dense-core input through Theorem~\ref{thm:R1_HS} yields the same vacuum-gap conclusion recorded here.
